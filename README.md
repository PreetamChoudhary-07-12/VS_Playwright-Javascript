# Antigravity Playwright Javascript

This repository follows my learning journey with JavaScript and Playwright automation. The code is organized by chapters as I cover new concepts.

## Project Structure

- **Chap1-Basics**: Introductory JavaScript concepts and setup.
- **Chap2-Basics**: JavaScript practices and Playwright basics.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/PreetamChoudhary-07-12/Antigravity_PlaywrightJavascript.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Antigravity_PlaywrightJavascript
   ```

3. **Run the scripts:**
   You can run the JavaScript files using Node.js:
   ```bash
   node Chap1-Basics/JS1-Basics.js
   node Chap2-Basics/JS1-Practice.js
   ```

## Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine.
- [Visual Studio Code](https://code.visualstudio.com/) recommended for editing.
